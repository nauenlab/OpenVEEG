OPENVEEG IS AN OPEN-SOURCE PROGRAM THAT ALLOWS TO RECORD EEG AND VIDEO OF SEIZURE-INDUCED MODEL ANIMALS.
In order to use OpenVEEG_Video, the user must independently install OpenCV 3.1 from http://opencv.org/, and include opencv_world.dll in the file folder containing the OpenVEEG_video executable.
Boost 1.64 or later must also be downloaded to use VideosStream. You can download it here: http://www.boost.org/.
Lab streaming layer can be found at https://github.com/sccn/labstreaminglayer.
